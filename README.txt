This is a simple language detection program that identifies common words within English and Spanish, then calculates the probability that a document is written in that language.

Simply download the file and run it. A command prompt window will open where you will need to specify a file to be read.
Input the filepath and the program will print out the total number of words, total paragraphs, and the language of the text.

Currently this only support English and Spanish documents, but could easily be updated for other languages.
